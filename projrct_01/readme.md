# 🖤 운치 (UNCHI) | 프리미엄 한국적 니치 향수 쇼핑몰

> **"어둠 속에 스며든 한국의 세 가지 운치"**
> 
> 서양식 향수 베이스에 한국 고유의 정서와 오브제(먹, 기와, 산사)의 기억을 한 방울 떨어뜨린 프리미엄 니치 향수 브랜드 **'운치(UNCHI)'**의 공식 웹사이트 프로젝트입니다. 애플과 토스의 미니멀하고 직관적인 UI/UX 디자인을 벤치마킹하여, 정적인 시각 에셋과 스토리텔링형 인터랙션만으로 극상의 브랜드 몰입감을 선사합니다.

<br>

## 📌 프로젝트 개요
* **프로젝트 유형:** 개인 포트폴리오 프로젝트 (웹 퍼블리싱 및 프론트엔드)
* **개발 기술:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **디자인 컨셉:** **Minimal & Bold** (다크 모드 기반의 고대비 레이아웃, 여백의 미, 우아한 타이포그래피)

---

## ✨ 핵심 기능 (Key Features)

### 1. 스토리텔링형 패럴랙스 스크롤 (Storytelling Scroll)
* 애플(Apple)의 제품 프레젠테이션 방식을 오마주하여, 동영상 없이 **고해상도 정지 이미지(Static Image)**와 Vanilla JS 코드로만 입체적인 화면 전환을 구현했습니다.
* 사용자의 스크롤 위치에 따라 시그니처 컬렉션 3종이 시차(Parallax) 효과, 페이드 트랜지션, 미세한 스케일 조정을 통해 한 편의 전시처럼 우아하게 등장합니다.

### 2. JSON 기반 동적 상품 목록 구현
* 정적인 하드코딩 방식에서 벗어나, 상품 데이터를 `products.json` 파일로 분리하여 관리합니다.
* JavaScript의 `fetch API`를 이용해 데이터를 동적으로 불러오며, 일반 상업 쇼핑몰처럼 사용자가 보기 편하고 직관적인 **그리드(Grid) 레이아웃**의 상품 목록 및 상세 섹션을 렌더링합니다.

### 3. 네비게이션(GNB) 및 싱글 페이지 인터랙션
* 4개의 핵심 서브 메뉴로 구성된 깔끔한 네비게이션 바를 상단에 고정(Sticky) 배치했습니다.
* 전체적인 사이트의 톤앤매너를 유지하면서, 각 서브 메뉴(상품 보기 등) 클릭 시 화면 전환 없이 해당하는 각각의 내용이 매끄럽게 표시되는 컴포넌트 스위칭 로직을 구현했습니다.

### 4. 실시간 커스텀 위젯 (날씨, 일자, 시간)
* 토스(Toss)의 대시보드처럼 직관적이고 깔끔한 디자인의 위젯을 상단에 배치했습니다.
* JavaScript 내장 함수를 활용한 실시간 날씨, 현재 일자 및 시간 정보를 받아와 화면에 동적으로 표기하여 사용자 맞춤형 경험을 제공합니다.

### 5. 카카오 맵(Kakao Map) API 연동 오시는 길
* 브랜드의 오프라인 쇼룸 위치를 안내하는 '오시는 길' 섹션에 카카오 맵 API를 연동했습니다.
* 다크 모드 웹사이트 분위기에 이질감이 들지 않도록 커스텀 마커와 심플한 맵 레이아웃을 적용했습니다.

## 🗺️ 정보 구조도 (Navigation & Menu)

상단 네비게이션은 총 **4개의 서브 메뉴**로 구성되어 있으며, 메뉴 클릭 시 JavaScript를 통해 각 섹션의 컨텐츠가 동적으로 활성화됩니다.

| 메뉴명 | 주요 내용 및 UI/UX 연출 |
| :--- | :--- |
| **BRAND STORY** | 브랜드 철학과 '스토리텔링형 스크롤'이 시작되는 메인 인트로 섹션 |
| **SHOP (상품 보기)** | `products.json` 데이터를 로드하여 바둑판식(Grid)으로 정렬된 상품 목록 표시 |
| **WIDGET** | 오늘 현재 날씨, 일자, 시간이 감각적인 텍스트로 표시되는 사용자 인터랙션 영역 |
| **SHOWROOM** | 카카오 맵 API를 활용한 오프라인 쇼룸 안내 및 오시는 길 내비게이션 |

## 🪵 시그니처 라인업 (Signature Trilogy)

서양식 베이스 위에 한국적 운치를 세련되게 담아낸 3가지 대표 상품을 토대로 쇼핑몰이 구성되어 있습니다.

```
├── 1. 묵향 (墨香) : 새벽 서재의 침묵 
│    └── 서늘한 새벽 공기 속, 오래된 고서가 가득한 서재에서 먹을 갈 때 퍼지는 차분하고 지적인 향
├── 2. 와우 (瓦雨) : 비 개인 기와 아래, 흙내음
│    └── 소나기가 지나간 뒤, 눅눅하게 젖은 한옥 기와와 이끼 낀 서까래에서 피어오르는 서정적인 흙내음
└── 3. 고사 (古寺) : 산사의 새벽안개와 향유
     └── 깊은 산속 때 묻지 않은 고찰의 차가운 새벽안개 뚫고 은은하게 번지는 신비로운 향의 기운
```


## 🛠️ 기술 스택 & 개발 지침 (Tech Stack)

* **UI/UX & Markup:** 웹 접근성 표준을 준수하는 시맨틱 마크업(`header`, `nav`, `section`, `footer` 활용)
* **CSS3 Flexbox / Grid:** 상품 목록 페이지의 정갈한 바둑판 배열 및 미니멀한 레이아웃 배치
* **Vanilla JavaScript:** 외부 라이브러리(jQuery 등) 없이 순수 스크립트로만 스크롤 제어 및 API 비동기 통신 구현

## 🚀 실행 방법 (How to Run)

1. 저장소를 클론합니다.
   ```bash

   ```
2. 프로젝트 폴더로 이동한 뒤 `index.html` 파일을 라이브 서버(Live Server)를 통해 실행합니다.
3. *주의: 카카오 맵 API 및 날씨 데이터 연동을 위해 로컬 서버 환경(VS Code Live Server 등)에서의 실행을 권장합니다.*
